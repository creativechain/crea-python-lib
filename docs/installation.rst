Installation
############

Install with `pip`:

::

    $ sudo apt-get install libffi-dev libssl-dev python-dev
    $ pip3 install dpay-lib dpay-cli

Manual installation:

::

    $ git clone https://github.com/dpays/dpay-python-lib
    $ cd dpay-python-lib
    $ python3 setup.py install --user

Upgrade
#######

::

   $ pip install --user --upgrade
