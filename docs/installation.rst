Installation
############

Install with `pip`:

::

    $ sudo apt-get install libffi-dev libssl-dev python-dev
    $ pip3 install crea-lib creacli

Manual installation:

::

    $ git clone https://github.com/creativechain/crea-python-lib
    $ cd crea-python-lib
    $ python3 setup.py install --user

Upgrade
#######

::

   $ pip install --user --upgrade
